# Smart Presentation Builder

> Auto-generate PowerPoint presentations from text outlines

## Overview
This project is part of a comprehensive AI/ML portfolio covering python automation.

## Features
- Production-ready implementation
- Well-documented code with examples
- Easy to customize and extend

## Installation

```bash
git clone https://github.com/jaychampaneri14/smart-presentation-builder.git
cd smart-presentation-builder
pip install -r requirements.txt
python main.py
```

## Requirements
See `requirements.txt` for full dependencies.

```bash
python main.py
```
